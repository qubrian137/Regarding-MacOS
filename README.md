# Regarding-MacOS

**Regarding-MacOS** is a software designed specifically for macOS.  
Developed using **SwiftUI**, it aims to deliver a seamless experience on Apple platforms.

The latest available update is **Pro 3.1.1** (as of 2026/2/17).  
Initially targeting iOS, the project is planned to migrate to an iOS & macOS compatible version (**Beta 1.9**) on **February 21, 2026**.
